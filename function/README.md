STRICT MODE
- strict mode is a special mode that we can activate in JavaScript, which makes it easier for us to write a secure JavaScript code
=> to use strict mode: `use strict`
=> everything before the lines code will not activated 

- strict mode help us introduce the bugs into our code for 2 reason
1. First, strict mode forbids us to do certain things
2. Second, it will actually create visible errors for us
